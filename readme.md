## 小程序之个人博客
> 最近，微信发布了小程序，一时间风靡整个朋友圈，一大批前端工作者们开始为之疯狂，但由于属于应用号开发，所以个人是没办法申请开发的，只能通过公司或者组织等机构申请，有幸，在公司得到了比较好的支持，申请到了小程序开发资格。开始我们的痛并快乐的挖坑之旅。


#### 项目简介
> 本项目是使用小程序开发模式，以一个博客APP为开发案例，尽可能涵盖小程序所有的组件和API，组装成一个完整的小程序案例。

#### 项目功能
1. 登录注册
2. 精彩博文推荐
3. 最新博文列表
4. 文章阅读，点赞，评论
5. 文章发布
6. 个人中心
7. 系统配置等
